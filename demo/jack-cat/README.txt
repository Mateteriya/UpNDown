Демо: Валет — Кот в шляпе (4 масти) + Кот в сапогах
====================================================

Структура
---------
  images/
    jack-cat-in-hat.png      — общий «кот в шляпе» (пока для всех мастей)
    jack-cat-hat-spades.png  — валет ♠ (когда будет готов)
    jack-cat-hat-hearts.png  — валет ♥
    jack-cat-hat-diamonds.png— валет ♦
    jack-cat-hat-clubs.png   — валет ♣
    jack-cat-face.png        — старый вариант (голова без шляпы)

  saved-for-other-projects/
    jack-puss-in-boots.png   — Кот в сапогах (для других проектов, не для этого)

Как посмотреть
--------------
  Откройте в браузере index.html.
  Вверху — 4 карты по мастям (голова кота ограничена ~⅓ карты по высоте).
  Ниже — блок «Сохранено для других проектов» с Котом в сапогах.

Требования к 4 вариантам «Кот в шляпе» (по мастям)
-------------------------------------------------
  • Только голова кота в шляпе (не полная фигура).
  • Явно кот-парень (не кошка).
  • Голова занимает примерно треть по высоте карты — в файле лучше рисовать голову небольшои по центру, с запасом пустого места сверху/снизу, чтобы при вставке в карту она не «размазывалась» на весь центр.
  • Отличие по мастям — цвет/стиль шляпы или акцент:
    ♠ Пики    — чёрная или тёмно-синяя шляпа, можно серебристая отделка
    ♥ Червы   — красная/бордовая шляпа, можно с лентой
    ♦ Бубны   — золотистая/жёлтая шляпа, можно с «бриллиантом»
    ♣ Трефы   — зелёная/изумрудная шляпа, можно с трилистником

Когда добавите 4 файла в images/ с именами jack-cat-hat-spades.png и т.д., демо само подхватит их (fallback на jack-cat-in-hat.png, если файла нет).

Встройка в приложение
---------------------
  После выбора/подготовки 4 картинок — положить в public/cards/ и в CardView.tsx для ранга J показывать <img> с нужным файлом в зависимости от card.suit.
  Кота в сапогах хранить в saved-for-other-projects/ и использовать в других проектах.

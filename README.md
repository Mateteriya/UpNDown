# Up&Down

Мультиплатформенное приложение для карточной игры на взятки «Up&Down».

## Документация

- [Техническое задание (TZ)](./TZ.md)
- [План дальше (6 шагов)](./docs/PLAN-DALEE.md) — фундамент выполнен; далее: мобильная полировка → онлайн → уровни ИИ → архив результатов → турниры/обучение
- [Правила для AI и разработчиков](./AGENTS.md)

## Разработка

```bash
# Установка зависимостей (в папке проекта)
npm install

# Запуск в режиме разработки
npm run dev

# Сборка
npm run build

# Тесты
npm run test
```

> **Примечание (Windows):** Папка `Up$Down` содержит символ `$`. В PowerShell используйте одинарные кавычки: `cd 'd:\Projects\Up$Down'`

## Структура проекта

```
├── src/
│   ├── game/          # Ядро игровой логики
│   │   ├── deck.ts    # Колода, карты
│   │   ├── rules.ts   # Правила игры
│   │   ├── scoring.ts # Подсчёт очков
│   │   └── types.ts   # Типы
│   ├── ai/            # Искусственный интеллект
│   └── ui/            # Пользовательский интерфейс
```

## Roadmap

1. **MVP** — ядро логики, офлайн против ИИ, базовый UI (Web) — **готово**
2. **Фундамент** — профиль (имя, аватар, profileId), рейтинг, авторизация (Supabase) — **готово**
3. **Альфа** — мобильная полировка, онлайн-матчмейкинг (лобби/столы), уровни ИИ
4. **Бета** — турнирный режим, архив результатов, мобильные приложения (APK)
5. **Релиз 1.0** — турниры, обучение, маркетинг

Подробный план шагов 2–6: [docs/PLAN-DALEE.md](./docs/PLAN-DALEE.md).

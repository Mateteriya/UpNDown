# Иконка PWA на рабочем столе Windows

Если при установке приложения из Chrome иконка в трее правильная, но **ярлык на рабочем столе без иконки**, это известное поведение Windows/Chrome.

## Что уже сделано в проекте

- В манифесте есть иконки 192×192, 256×256 и 512×512.
- При сборке из `icon-256.png` генерируется `favicon.ico` (формат .ico лучше подходит для Windows).
- В `index.html` подключён `favicon.ico` первым.

После деплоя и **повторной установки** PWA (удалить приложение → установить снова) иконка на рабочем столе может начать отображаться.

## Если иконка на рабочем столе всё равно пустая

### 1. Назначить иконку вручную

1. Создайте на диске файл иконки в формате **.ico** (например, конвертируйте `public/icon-256.png` в .ico через [convertio.co](https://convertio.co/ru/png-ico/) или другой сервис).
2. Правый клик по ярлыку на рабочем столе → **Свойства**.
3. Кнопка **«Сменить значок»** (Change Icon).
4. **«Обзор»** → укажите сохранённый .ico файл → OK.

Иконка ярлыка будет браться из этого файла и не пропадёт.

### 2. Профиль Chrome на сетевом диске

Если папка профиля Chrome (или часть пути к ней) лежит на сетевом/виртуальном диске, Windows может не показывать иконки таких ярлыков. Варианты:

- Перенести профиль Chrome на локальный диск, или
- Включить загрузку иконок с удалённых путей (только для опытных пользователей):
  - Запустить **cmd от имени администратора**.
  - Выполнить:
    ```
    reg add "HKLM\SOFTWARE\Policies\Microsoft\Windows\Explorer" /v EnableShellShortcutIconRemotePath /t REG_DWORD /d 1 /f
    ```
  - Перезагрузить ПК.

### 3. Сброс кэша иконок Windows

Иногда помогает сброс кэша значков:

- Закрыть все окна проводника.
- Удалить (или переименовать) файл кэша иконок (путь может отличаться в зависимости от версии Windows), затем перезагрузить ПК. Либо воспользоваться утилитами очистки кэша иконок из интернета.

Если после этого ярлык по-прежнему без иконки, самый надёжный вариант — **п. 1 (назначить иконку вручную)**.

# Сервер игры в вашей Wi‑Fi (ПК)

Программа на компьютере: **создать комнату**, показать **QR** или **код**. Игроки подключаются с телефонов **в той же Wi‑Fi**.

Работает **только в одной сети**. Игра через интернет с другого города — облачный сервер на VPS (не эта утилита).

---

## Для пользователя (установщик с сайта)

Скачайте и установите **«Up&Down — игра в сети»** (Windows). Подробно: [HOST-INSTALLER.md](./HOST-INSTALLER.md).

На ПК нужны только **Windows**, **Wi‑Fi** и **разрешение брандмауэра**. Node.js и терминалы **не нужны**.

---

Полный гайд «когда Rust, как выложить на сайт, что дальше с VPS»: [HOST-LAN-PUBLISH.md](./HOST-LAN-PUBLISH.md).

## Для разработки (пока нет сайта)

| Нужно | Зачем |
|--------|--------|
| **Node.js 18+** | Запуск сервера |
| **npm install** | Один раз в корне репозитория |
| **Wi‑Fi** | ПК и телефоны в одной сети |
| **Брандмауэр** | «Разрешить доступ» для Node при первом запуске |

**Не нужно:** VPN, Supabase, cloudflared, ngrok.

### Быстрый запуск (одна команда)

```powershell
cd D:\Projects\UpNDown
npm run build:host-game
npm run host:app
```

Откроется панель `http://127.0.0.1:3001/host` → **Создать комнату** → QR / Поделиться.

Если панель «старая»: `npm run host:kill`, затем снова `npm run host:app`, в браузере **Ctrl+F5**.

### Сборка установщика .exe

```powershell
npm run host:installer:prep
npm run host:desktop:build
```

См. [HOST-DESKTOP.md](./HOST-DESKTOP.md) и [HOST-INSTALLER.md](./HOST-INSTALLER.md).

## На телефонах гостей

- Та же **Wi‑Fi**, что у ПК.
- Открыть **QR** или ссылку (в ВК иногда удобнее **скопировать ссылку** и вставить в Chrome/Safari).
- По ссылке с панели — экран **«Вход в комнату»** и автоподключение (`autojoin=1`).
- Галочка **«Показывать стол в „Зале столов“»** на панели (по умолчанию включена) — стол виден в зале на телефонах.

---

## Запасные порты

В панели — альтернативные ссылки (порты **3002**, **3003**), если основной не открывается.

---

## Сценарий

1. ПК: запуск панели → **Создать комнату**.
2. Телефоны: QR или «Поделиться» → **Войти в комнату**.
3. Хост может играть с телефона по той же ссылке.

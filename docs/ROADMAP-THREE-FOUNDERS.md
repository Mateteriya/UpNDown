# Roadmap для троих: приоритеты, роли, сроки

Одностраничный ориентир для **владельца продукта**, **технического директора** и **партнёра по международному развитию**. Детали по приложению (WS, IAP, CC) — [APP-WORKPLAN-WS-IAP-CC.md](./APP-WORKPLAN-WS-IAP-CC.md). Офлайн и маркетинг — отдельные треки ниже; юридические формулировки Cash Arena — [PARTNERSHIP-WAVE5-TRIGGERS.md](./PARTNERSHIP-WAVE5-TRIGGERS.md).

---

## 1. Четыре слоя продукта (не путать с «четырьмя направлениями работ»)

| Слой | Содержание | Нужна gambling-лицензия? |
|------|------------|----------------------------|
| **1. Игра** | skill, рейтинг, друзья, онлайн | нет |
| **2. Продажи в приложении** | косметика, ИИ, 3 игрока, подписка | нет (IAP) |
| **3. Cosmic Credits (CC)** | виртуальная валюта, турниры, **без cash-out** | нет (при корректных ToS) |
| **4. Real money** | Cash Arena, KYC, whitelist стран | да, **точечно** |

Слои 1–3 — **первая очередь** и основной доход на старте. Слой 4 и выбор юрисдикции для онлайн-платежей — **вторая очередь**, когда есть продукт, аудитория и юрист.

## 2. Роли (кто за что отвечает)

| Роль | Зона ответственности |
|------|---------------------|
| **Владелец продукта** | видение, UX, игровой дизайн, приоритеты roadmap, бренд, контент, координация; решения по Грузии (офлайн-пилот); финальный go/no-go по фичам |
| **Технический директор** | игровой WS-сервер, VPS, CI/deploy сервера, стабильность онлайна, интеграция app ↔ инфра; оценка сроков по backend; при необходимости — юрлицо/хостинг в РФ для **не-gambling** сервисов |
| **Партнёр по международному развитию** | entity за рубежом (волна 5+), compliance, lawyer liaison, EN-рынок, PSP-переговоры, MOU; **не** подменяет продуктовые решения владельца |

Созвон и чеклист для партнёра: [PARTNERSHIP-AU-CALL-GUIDE.md](./PARTNERSHIP-AU-CALL-GUIDE.md) (название файла историческое; в переписке — «международный партнёр»).

## 3. Первая очередь: четыре направления

| ID | Направление | Владелец | Со-исполнители | Заметка |
|----|-------------|----------|----------------|---------|
| **а** | **Офлайн-якорь (Грузия)** | Владелец продукта | Международный партнёр (переговоры EN), местный юрист | Пилот клуба / оператор с лицензией; крупные вложения; **не блокирует** б, в, г |
| **б** | **Продажный слой** (CC, косметика, ИИ, 3 игрока) | Владелец продукта | Техдиректор (API, флаги) | Основной легальный cashflow без gambling-лицензии — см. [APP-WORKPLAN-WS-IAP-CC.md](./APP-WORKPLAN-WS-IAP-CC.md) |
| **в** | **Игровой сервер (WebSocket)** | Технический директор | Владелец продукта (тест, URL в env) | Замена тяжёлого Supabase-polling в партии; Supabase остаётся: auth, профили, архив, CC |
| **г** | **Маркетинг** (офлайн + онлайн) | Владелец продукта | Международный партнёр (EN-каналы), техдиректор (лендинг/метрики) | Срочно и **параллельно** с б и в; без г офлайн не окупится |

**Параллельный старт:** б + в + г. **а** — с первым звонком местному юристу и поиском партнёра площадки, пока г готовит спрос.

## 4. Вторая очередь

| ID | Направление | Владелец | Когда имеет смысл |
|----|-------------|----------|-------------------|
| **2a** | **Юрисдикция онлайн-денег** (entity + PSP, KYC) | Международный партнёр + lawyer | После стабильных слоёв 1–3, метрик, офлайн-кейса |
| **2b** | **Лицензии для онлайн-cash** | Lawyer + entity | Только для **Cash Arena** в целевых странах whitelist; **не** для CC, IAP, «игра с друзьями» |

Офлайн-лицензия / оператор в **Грузии** относится к направлению **а**, а не заменяет **2a/2b** для глобального online gambling.

## 5. Ориентир по срокам (Q2–Q4 2026)

Сегодня — конец **Q2 2026**. Сроки ориентировочные; сдвиг на 2–4 недели нормален.

### Q2 2026 (остаток) — «фундамент и запуск параллельных треков»

| Кто | Фокус |
|-----|--------|
| Владелец | Полировка мобильного стола; включить публичный зал при необходимости (`VITE_PUBLIC_HALL_ENABLED`); старт **г** (контент-план, первые события/комьюнити); brief юристу по Грузии |
| Техдиректор | Договориться о VPS; локальный прототип WS; оценка миграции с Supabase-play |
| Международный партнёр | Закрыть итоги созвона (Go/Soft/No); черновик MOU при Go; не трогать cash до триггеров волны 5 |

### Q3 2026 — «WS в проде + первые продажи»

| Кто | Фокус |
|-----|--------|
| Владелец | Магазин / IAP (косметика), гейт ИИ и 3 игрока; UI кошелька CC; бета-тест с друзьями |
| Техдиректор | `wss://` на VPS, мониторинг, протокол стабилен; опционально — лёгкий egress-tick в Supabase до полного переезда партии |
| Все | Пилот офлайн в Грузии (1–2 вечера), если юрист + партнёр готовы |

### Q4 2026 — «масштаб social + CC, решение по волне 5»

| Кто | Фокус |
|-----|--------|
| Владелец | CC daily grant, турниры sit-n-go (флаг); ASO / сторы; повторяемый офлайн-формат |
| Техдиректор | Нагрузка 20–40 комнат, бэкапы, алерты |
| Международный партнёр | При выполнении триггеров — entity, lawyer, PSP (см. wave 5 doc) |

**2027+:** Cash Arena только после sign-off lawyer; не смешивать с массовым social.

## 6. Сознательно не делаем (на старте)

- Массовый **online real-money** «для всех стран» без KYC и whitelist.
- **Cash-out** Cosmic Credits в фиат или крипту.
- Позиционирование продукта как казино; смешение CC и «живых денег» в одном UI без разделения.
- Обход регуляторов или «серые» схемы приёма ставок там, где нет лицензии.
- Ставка только на один канал (только офлайн или только онлайн без маркетинга).
- Регистрация gambling-entity **до** работающего social-слоя и хотя бы одного стабильного канала монетизации (IAP / CC).
